To use this program, you have to type this prompt on the terminal: "make", this will compile the files to see that we don't have any error or warning and create the objects files (.o) and de executable to start running the program. 
Once you execute the command "make" to create the executable, you have to start running the program prompting this command "./autos.exe [Number of record]" on the terminal, its important to type this number after the executable so the program can initiate, in case is not specified as the second argument on the terminal, its going to give a "segmentation fault ( core dumped)" error.
Once you started running the program, you have to record the data of your vehicles so, its important to type the plate with a string of 6 character where the last 2 are numbers, this will give the necessary information to the program and give you a range of a date that you can go and verify your car.
This will create a .txt file that has the recorded vehicle data, this information swill stay in the file until you execute the program and record new vehicle data.
